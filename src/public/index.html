FaviconAPI

Get the highest resolution favicon for any domain - or look up a service icon by name.

Favicons are resolved from an HTML scraper and ten upstream providers: Google, DuckDuckGo, Yandex, Favicon.so, Vemetric, Faviconextractor, FaviconKit, favicon.run, logo.dev, and Brandfetch. App icons are resolved from selfh.st, dashboardicons.com, LobeHub, and SVGL.

⚙ Tip Use the favicon directly via https://<this-host>/{domain-or-appname} (e.g. reddit.com, jellyfin, immich). Click any image to copy its URL.
Try: github.com reddit.com proton.me hosthatch.com google immich bitwarden

Browser tools

Shortcuts to use FaviconAPI from your browser address bar or bookmarks bar.

🔍 Browser search

Add FaviconAPI as a custom search engine and look up favicons straight from your address bar.

🔗 Custom URL

Build a shareable URL with your own preferred provider, fallbacks and minimum icon size. The settings are encoded in the URL — no account needed.

★ Bookmarklet

Drag the button to your bookmarks bar (don't click & copy — the name will only auto-fill when dragging). Click the saved bookmark on any page to copy that site's favicon URL.

Search from browser

Add FaviconAPI as a custom search engine. Use %s as the query placeholder — your browser replaces it with whatever you type in the address bar.

http://localhost:3100/search?q=%s

Google Chrome

  1. Open Settings → Search engine → Manage search engines and site search.
  2. Click Add under Site search.
  3. Name: FaviconAPI
  4. Shortcut: fav (optional)
  5. URL:

Microsoft Edge

  1. Open Settings → Privacy, search, and services → Address bar and search → Manage search engines.
  2. Click Add.
  3. Name: FaviconAPI
  4. Keyword: fav (optional)
  5. URL:

Mozilla Firefox

  1. Firefox may offer Add search engine automatically on this page (OpenSearch).
  2. Or open Settings → Search → scroll to Search Shortcuts → Add.
  3. Name: FaviconAPI
  4. Keyword: fav (optional)
  5. URL:

Apple Safari

  1. Desktop Safari does not support custom search-engine URLs.
  2. On iPhone/iPad: Settings → Safari → Search Engine (only built-in engines are available).

Use Chrome, Edge or Firefox for this feature, or use the bookmarklet in Tools.
